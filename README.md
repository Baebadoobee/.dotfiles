## Instalation

First, clone this repository

```sh
git clone https://github.com/Baebadoobee/.dotfiles.git ~/.dotfiles && cd ~/.dotfiles
```

Then, exec the `instal.sh` script

```sh
chmod +x ./install.sh
./install.sh
```

After this, you can select your desired dot.
